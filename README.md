# Changelog

All notable changes to ETEMENAKI are documented here.

## [1.0.0] — 2026-03-13

### Added
- Complete 50-level learning platform in a single HTML file
- 5 tiers: Foundation, Combination, Real-World, Advanced, Mastery
- 200 practice exercises with keyword-specific validation
- 200 pattern recognition games
- 50 master challenges
- 250 assessment questions
- Tower of Babel puzzle system with jigsaw pieces and secret message
- XP system, streak counter, toast notifications, screen flash effects
- Speed run mode with personal best tracking
- Responsive design for desktop and tablet
- Hub view with level select grid organized by tier
- Level progression: complete one to unlock the next

### Level Content Depth
- Levels 1–19: 4 fully unique hand-authored examples per level
- Levels 20–30: 2 unique examples + 2 derived variations per level
- Levels 31–33: 4 fully unique hand-authored examples per level
- Levels 34–50: 2 unique examples + 2 variations per level
- All 200 practice validators use topic-specific keyword checks (zero weak validators)

### Architecture
- Zero dependencies — single HTML file, ~426 KB
- Pure JavaScript, no frameworks
- Canvas-rendered puzzle with 7-tier ziggurat Tower of Babel
- JSON-based level data for easy content updates

## [0.1.0] — 2026-01-16

### Added
- Level 1 prototype: Storage & Types
- 6 example-practice pairs
- Pattern recognition games
- Mini challenge and assessment
- Replay/speed run system
- Non-intrusive gamification (toasts, XP popups, screen flash)
- Personal best tracking via localStorage
- Multiple test files for browser debugging
- Project architecture document (v1.0)
- Complete session documentation
